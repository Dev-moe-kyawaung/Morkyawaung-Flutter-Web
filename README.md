## ✨ Features

### 🎯 Core Features
- **Cyber-Neon Design** - Neon Cyan + Hot Pink + Electric Yellow
- **Animated Grid Background** - CRT scanlines + glow orbs
- **Sticky Navbar** - Scroll-aware glow, Myanmar labels, mobile hamburger
- **Hero Section** - Typing effect with 5 rotating roles
- **Animated Stats** - Flip counters (8+ yrs, 15+ apps, 200K+ users, 10+ clients, 1,222 repos, 100%)
- **Skills Section** - Progress bars animate on scroll
- **Services Cards** - 6 cards with light-beam hover effect
- **Projects Gallery** - 15 app cards with tags + GitHub links
- **Certificates** - 6 certificates with glow borders
- **Contact Form** - Validation with Myanmar error messages
- **Burmese Support** - Full Burmese translations
- **Smooth Scroll** - Parallax depth effects

## 🚀 Getting Started

### Prerequisites
- Flutter SDK >= 3.10.0
- Dart SDK >= 3.0.0
- Chrome (for web)
- Firebase CLI (for deployment)

### Installation

```bash
# Clone repository
git clone https://github.com/moekyawaung13721/premium-flutter-portfolio.git
cd premium-flutter-portfolio

# Install dependencies
flutter pub get

# Run on Chrome
flutter run -d chrome

# Build for web
flutter build web --release

# Deploy to Firebase
firebase deploy --only hosting
